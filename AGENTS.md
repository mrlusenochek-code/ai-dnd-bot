# AGENTS.md

## Проект
ИИ ДНД бот / веб-сервер. Цель: исправная работа WebSocket и нормальное логирование.

## Как запускать
- Python venv
- pip install -r requirements.txt
- запуск: python -m uvicorn app.web.server:app --host 127.0.0.1 --port 8000 --reload

## Правила изменений
- Любые изменения — через git (отдельные коммиты).
- Не трогать секреты (.env, токены).
- Перед сдачей результата: запуск сервера или тестов (что найдёшь в проекте).
